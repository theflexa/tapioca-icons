# Regras do Projeto

## Comunicacao antes de acao

- Quando o usuario fizer uma **pergunta ou tiver duvida**, responda e apresente alternativas. NAO saia desenvolvendo ou implementando nada.
- So implemente codigo quando o usuario **explicitamente pedir** para fazer, criar, implementar ou corrigir algo.
- Na duvida, pergunte: "Quer que eu implemente isso ou esta apenas perguntando?"
